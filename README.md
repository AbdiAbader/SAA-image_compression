## Introduction

``` This project implements an image compression algorithm using Python and Flask.```

### Features

- [x] User can upload an image and specify the compression level
- [x] User can download the compressed image
- [x] The compressed image is stored in a temporary folder and deleted after download

## Installation

- Flask
- Pillow

- Angular CLI
